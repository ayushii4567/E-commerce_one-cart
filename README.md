# 🛒 AI-Powered E-Commerce Platform (MERN Stack)

## 🚀 Overview
A full-stack AI-powered e-commerce web application built using the MERN stack with secure authentication, payment integration, and a complete admin dashboard. The platform delivers a modern shopping experience with intelligent product recommendations, seamless checkout flow, and real-time management features.

This project demonstrates production-level architecture including authentication, role-based access, payment gateway integration, and AI-enhanced user experience.

---

## ✨ Key Features

### 🧑‍💻 User Features
- Modern and responsive UI built with React.js  
- Google Authentication (Firebase / Google OAuth)  
- Secure login & registration system  
- Browse products with detailed pages  
- Add to Cart / Remove from Cart functionality  
- Seamless checkout process  
- Razorpay payment integration for secure transactions  
- Order placement & order history tracking  
- AI-powered product recommendations  

---

### 🛠️ Admin Features
- Secure Admin Dashboard  
- Add / Edit / Delete Products  
- Manage Users  
- Manage Orders  
- Real-time stock updates  
- Product inventory control system  

---

### 🤖 AI Features
- Intelligent product recommendation system  
- Smart navigation assistance for users  
- Personalized product suggestions based on user activity  

---

## 🏗️ Tech Stack

### Frontend
- React.js  
- Redux / Context API  
- Tailwind CSS / CSS3  
- React Router  

### Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT Authentication  
- Google OAuth  

### Payment & Services
- Razorpay Payment Gateway  
- Firebase Authentication (Google Login)  

### Deployment
- Frontend: Vercel / Netlify  
- Backend: Render / Railway  
- Database: MongoDB Atlas
